This folder stores all the experiments and files created in the development of the hybrid patent clusters.

Important folders are:
1. cpc_codes: Folder for how we processed the current cpc/ipc codes for patents, along with the associated text
2. embedding_experiments: Folder containing experiments for finding embeddings with different models, including computing costs.
3. evaluation_metrics: Some of the small scale experiments and testing
4. faiss: Implementing the faiss algorithm, and updating the faiss dag
5. sql: Old sql scripts used for extracting evaluation metrics during development
